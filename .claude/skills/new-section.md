## Steps

1. Ask for: section name, purpose (hero/features/CTA/etc.), and any specific layout notes.
2. Invoke the `content-writer` agent to draft headline, subheadline, and CTA copy for this section.
3. Create the component file in the appropriate directory (`src/components/` or `src/sections/`).
4. Create a companion style file if the project uses CSS modules or scoped styles.
5. Import and add the section to the relevant page.
6. Confirm it renders without errors in the dev server.
